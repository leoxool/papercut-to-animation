## ✅ 完整功能列表

### 1. 班级管理

#### 创建班级
- **位置**: 左侧班级管理区域顶部
- **按钮**: "➕ 创建新班级" (青色)
- **功能**:
  - 设置班级名称（必填）
  - 设置班级描述（可选）
  - 选择班主任（从现有教师中选择，可选）

#### 编辑班级
- **位置**: 每个班级卡片右上角
- **按钮**: "编辑" (灰色)
- **功能**:
  - 修改班级名称
  - 修改班级描述
  - 更换班主任
  - 点击班级卡片其他区域可进入学生列表

#### 删除班级
- **位置**: 每个班级卡片右上角
- **按钮**: "删除" (红色)
- **保护机制**:
  - 班级人数 > 0 时按钮自动禁用
  - 鼠标悬停显示提示："该班级下还有学生，无法删除"
  - 点击禁用按钮会弹出警告

#### 查看学生列表
- **操作**: 点击班级卡片的空白区域
- **功能**: 进入该班级的学生列表视图
- **返回**: 点击顶部"← 返回班级列表"按钮

### 2. 用户管理

#### 批量创建用户
- **位置**: 右侧用户管理区域顶部
- **按钮**: "➕ 批量创建" (绿色)
- **功能**:
  1. **选择注册类型**: 学生 / 教师 / 管理员
  2. **输入用户名列表**: 多用户名用逗号分隔
     - 示例：`张三,李四,王五`
  3. **选择班级**: 如果是学生，必须选择班级
  4. **默认设置**:
     - 密码：`12345678`
     - 真实姓名：与用户名相同
     - 账户状态：已激活

#### 管理员列表
- **位置**: 右侧用户管理区域上部
- **显示内容**:
  - 管理员姓名和用户名
  - 紫色"管理员"标签
- **操作**:
  - **重置密码**: 蓝色按钮，重置密码为 `12345678`
  - **删除**: 红色按钮（默认管理员 LEO 不能删除）

#### 教师列表
- **位置**: 右侧用户管理区域下部
- **显示内容**:
  - 教师姓名和用户名
  - 蓝色"教师"标签
- **操作**:
  - **重置密码**: 蓝色按钮，重置密码为 `12345678`
  - **删除**: 红色按钮

### 3. 学生列表视图

#### 查看学生
- **入口**: 点击班级卡片进入
- **显示内容**:
  - 班级名称和描述
  - 学生总数
  - 学生列表（姓名、用户名、创建时间）

#### 管理学生
- **添加学生**: 点击"➕ 添加学生"按钮
- **编辑学生**: 点击"编辑"按钮
  - 可修改学生姓名
  - 可更改学生所属班级
  - 可启用/禁用账户
- **重置密码**: 点击"重置密码"按钮（蓝色）
  - 重置密码为 `12345678`
- **删除学生**: 点击"删除"按钮（红色）

## 🔐 密码管理

### 重置密码功能
适用于所有用户（管理员、教师、学生）

**操作流程**:
1. 点击用户条目右侧的"重置密码"按钮（蓝色）
2. 弹出确认对话框：
   ```
   确定要重置用户 [用户名] 的密码吗？
   新密码将为：12345678
   ```
3. 确认后密码立即重置
4. 显示成功提示

**特性**:
- ✅ 一键重置，无需记住复杂密码
- ✅ 统一密码，便于管理
- ✅ 确认机制，防止误操作
- ✅ 即时生效，旧密码立即失效

## 🚀 使用流程示例

### 示例1：批量创建学生
1. 点击右侧"➕ 批量创建"按钮
2. 选择"学生"
3. 输入用户名：`张三,李四,王五,赵六`
4. 选择班级："艺术1班"
5. 点击"批量创建"
6. 提示"成功创建 4 个用户！"
7. 查看右侧教师列表，新学生出现在班级中

### 示例2：重置教师密码
1. 在右侧教师列表中找到目标教师
2. 点击该教师条目右侧的"重置密码"按钮（蓝色）
3. 确认对话框中点击"确定"
4. 提示"教师 [用户名] 的密码已重置为：12345678"
5. 教师可使用新密码登录

### 示例3：编辑班级信息
1. 在左侧班级列表中找到目标班级
2. 点击该班级卡片右上角的"编辑"按钮
3. 在弹出的编辑表单中修改信息
4. 点击"保存"
5. 班级信息立即更新

### 示例4：查看班级学生
1. 点击班级卡片的空白区域（或班级名称）
2. 进入学生列表视图
3. 查看该班级的所有学生
4. 点击"← 返回班级列表"回到主页

## ⚠️ 注意事项

### 删除保护
- **班级**: 有学生的班级不能删除
- **管理员**: 默认管理员 LEO 不能删除
- **学生**: 删除前会弹出确认对话框

### 密码安全
- 重置后的密码统一为 `12345678`
- 建议用户首次登录后立即修改密码
- 重置操作不可撤销

### 批量操作
- 批量创建的用户真实姓名与用户名相同
- 如果批量创建学生，必须选择班级
- 批量操作不可撤销
