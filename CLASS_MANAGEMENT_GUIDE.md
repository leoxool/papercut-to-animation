# 管理员班级管理功能使用指南

## 功能概述

管理员班级管理功能允许管理员创建、编辑、删除班级，并为学生分配班级。

## 后端API接口

### 班级管理API

#### 1. 创建班级
```
POST /api/classes
Authorization: Bearer <token>

Body:
{
  "name": "班级名称",
  "description": "班级描述（可选）",
  "teacherId": "教师ID（可选）"
}

Response:
{
  "message": "班级创建成功",
  "class": { ... }
}
```

#### 2. 获取班级列表
```
GET /api/classes
Authorization: Bearer <token>

Response:
{
  "classes": [
    {
      "id": "班级ID",
      "name": "班级名称",
      "description": "班级描述",
      "teacherId": "教师ID",
      "studentCount": 学生数量,
      "createdAt": "创建时间",
      "updatedAt": "更新时间"
    }
  ]
}
```

#### 3. 更新班级
```
PUT /api/classes/:classId
Authorization: Bearer <token>

Body:
{
  "name": "新班级名称",
  "description": "新班级描述",
  "teacherId": "新教师ID"
}

Response:
{
  "message": "班级信息已更新",
  "class": { ... }
}
```

#### 4. 删除班级
```
DELETE /api/classes/:classId
Authorization: Bearer <token>

Response:
{
  "message": "班级已删除"
}
```

⚠️ **注意**：如果班级下还有学生，删除操作会失败。

## 前端界面功能

### 管理员控制台

管理员登录后会自动跳转到管理员控制台，包含两个主要标签页：

#### 1. 用户管理标签页
- 查看所有用户列表（管理员、教师、学生）
- 创建新用户（支持管理员、教师、学生三种角色）
- 编辑用户信息
- 删除用户
- 为学生分配班级

#### 2. 班级管理标签页
- 查看所有班级列表
- 显示每个班级的学生数量
- 创建新班级
- 编辑班级信息（名称、描述、班主任）
- 删除班级

### 创建班级

1. 点击"创建新班级"按钮
2. 填写班级信息：
   - **班级名称**：必填，需唯一
   - **班级描述**：可选
   - **班主任**：可选，可从现有教师中选择
3. 点击"创建"按钮完成创建

### 编辑班级

1. 在班级列表中找到要编辑的班级
2. 点击"编辑"按钮
3. 修改班级信息
4. 点击"保存"按钮

### 删除班级

1. 在班级列表中找到要删除的班级
2. 点击"删除"按钮
3. 确认删除

⚠️ **注意**：如果班级下还有学生，删除操作会被阻止。

## 使用流程示例

### 示例1：创建班级并分配学生

1. 使用管理员账户登录
2. 切换到"班级管理"标签页
3. 点击"创建新班级"
4. 输入班级名称"艺术1班"，描述"艺术创作班"
5. 点击"创建"
6. 切换到"用户管理"标签页
7. 点击"创建新用户"
8. 创建学生账户，在"班级"字段输入"艺术1班"
9. 保存后，学生自动分配到"艺术1班"

### 示例2：为班级分配班主任

1. 管理员登录
2. 在"用户管理"中创建教师账户
3. 在"班级管理"中创建班级时选择该教师作为班主任
4. 或编辑现有班级，选择教师作为班主任

## 数据关系

### 用户与班级的关系
- 学生通过 `className` 字段与班级关联
- 一个班级可以有多个学生
- 一个学生只能属于一个班级

### 教师与班级的关系
- 班级通过 `teacherId` 字段与教师关联
- 一个教师可以管理多个班级
- 一个班级最多有一个班主任

## 注意事项

1. **内存存储**：当前使用内存存储，重启服务器数据会丢失
2. **权限控制**：所有班级管理操作需要管理员权限
3. **数据验证**：
   - 班级名称不能为空且需唯一
   - 删除班级前需先移除所有学生
   - 指定班主任时需确保该用户存在且角色为教师

## 错误处理

常见错误及解决方法：

1. **"班级名称已存在"**
   - 解决：使用不同的班级名称

2. **"该班级下还有学生，无法删除"**
   - 解决：先删除或移动班级下的学生

3. **"指定的教师不存在"**
   - 解决：确保教师ID正确且用户角色为教师
