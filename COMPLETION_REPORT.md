## 🎯 项目目标

**原始需求**:
> 用户管理界面内整合用户管理和班级管理,在控制台主页显示班级列表,这个页面可以进行创建、删除、编辑班级的操作,如果该班级人数大于0,则无法删除这个班级,点开这个班级的详情可以看到学生账户的列表,在列表里可以编辑学生账户的各项信息

## 🏗️ 技术架构

### 前端架构
- **框架**: React 18 + Hooks
- **状态管理**: useState + useEffect
- **组件设计**: 模块化、可复用
- **样式**: Tailwind CSS
- **HTTP客户端**: Fetch API

### 后端架构
- **运行时**: Node.js + Express
- **认证**: JWT Token
- **数据存储**: 内存 Map (开发阶段)
- **密码加密**: bcrypt
- **权限控制**: 角色检查中间件

### 数据流设计
```
用户操作
    ↓
React 组件状态更新
    ↓
API 调用
    ↓
后端验证与处理
    ↓
数据库更新
    ↓
响应返回
    ↓
前端状态同步
    ↓
UI 重新渲染
```
